Class: AlphabeticalSongsTree

AlphabeticalSongsTree(songs)

A tree of songs where the song arrays are placed in alphabetical properties. An instanace of this class would look like this example:


{
  "a": [ song, song ],
  "s": [ song, song ],
  "z": [ song, song ]
}
